A **tag type** is a group of related tags, while a **tag** is an instantiation of a tag type. Examples illustrate this best.

You might create a `hardware` tag type with a tag for each of the hardware types that your robots run on: `Raspberry Pi 4 Model B`, `NVIDIA Jetson Orin Nano`, and `NVIDIA Jetson Orin AGX`.

<img height="200" src="/images/vocabulary/hardware-tag-list.png" style={{borderRadius: '7px'}}/>

Or you could create an `accelerometer` tag type with a tag for each accelerometer: `Accelerometer 1`, `Accelerometer 2`, `Accelerometer 3`, `Accelerometer 4`, etc. 

<img height="200" src="/images/vocabulary/accelerometer-tag-list.png" style={{borderRadius: '7px'}}/>

Or you could create an `environment` tag type with tags for different stages of development: `Dev`, `QA`, `Beta`, and `Stable`.

<img height="200" src="/images/vocabulary/environment-tag-list.png" style={{borderRadius: '7px'}}/>

Miru allows you to "tag" clients with a particular tag, providing a simple way to group clients together logically.
