Any tags of the `hardware` tag type can only store metadata that adheres to the `hardware` metadata schema.

Metadata schemas look eerily similar to config schemas but they serve different purposes. A metadata schema does not describe or enforce any constraints on the concrete configs that are consumed by clients. Metadata schemas are only used for describing what constitutes valid metadata, effectively structuring the metadata stored for clients and tags.
