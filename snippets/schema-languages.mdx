Miru only supports JSON Schema, which is widely used and has [code generation](https://quicktype.io/) for many languages. However, support for other configuration languages is coming soon. Here are some popular ones.
- [CUE Language](https://cuelang.org/)
- [Pkl](https://pkl-lang.org/index.html)
- [Type Schema](https://typeschema.org/)
- [JSON Type Definition](https://jsontypedef.com/)
- [Dhall](https://dhall-lang.org/)
