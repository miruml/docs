Click the **Activate** button on the device you just created.

<Frame>
   ![](https://assets.miruml.com/docs/images/devices/one-device-to-activate.png)
</Frame>

This will open a dialog containing a unique, pre-authenticated installation command. Copy this command - you'll need it in the next step.

<Frame>
    ![](https://assets.miruml.com/docs/images/devices/activate-dialog.png)
</Frame>

<Warning>
The activation token in this install command is short-lived and device-specific. Don't share it or reuse it for other devices.
</Warning>
