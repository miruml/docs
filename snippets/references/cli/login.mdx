To log in, run the `login` command.

```bash
miru login
```

Retrieve your authentication token from the [Secrets](https://app.mirurobotics.com/settings/cli-token) page.

<Warning>
  You must be an <Tooltip tip={ADMIN_TOOLTIP.tip} cta={ADMIN_TOOLTIP.cta} href={ADMIN_TOOLTIP.href}>admin</Tooltip> user to retrieve an authentication token.
</Warning>
