To login, run the `login` command.

```bash
miru login
```

Retrieve your authentication token from the [Secrets](https://app.miruml.com/settings/cli-token) page.

<Warning>
    You must be an `admin` user to retrieve an authentication token.
</Warning>

    <img src="https://assets.miruml.com/docs/v02/images/secrets/cli-token/page.png" style={{borderRadius: '16px'}} />
</Frame>
