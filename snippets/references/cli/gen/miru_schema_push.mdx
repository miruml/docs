### Command

```bash
miru schema push PATH... [OPTIONS]
```

### Description

Upload one or more config schemas to miru.


### Examples


```bash Single Schema
miru schema push ./path/to/config/schema.yaml
```


```bash Multiple Schemas
miru schema push \
	./path/to/config/schema1.yaml \
	./path/to/config/schema2.yaml
```

