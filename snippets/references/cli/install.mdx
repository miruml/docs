Run the installation command in your local machine's terminal.

<Tabs>
    <Tab title="Linux">
        ```bash
        curl -fsSL https://raw.githubusercontent.com/miruml/cli/main/install.sh | sh
        ```
        The script requires `curl`, `tar`, `grep`, and `cut` to be installed.
        Review the contents of the installation script [here](https://github.com/miruml/cli/blob/main/install.sh). 
    </Tab>

    <Tab title="macOS">
        ```bash
        brew install miruml/cli/miru
        ```
    </Tab>
</Tabs>

<Note>
The Miru CLI supports **macOS** and **Linux**. Windows is not supported.
</Note>
