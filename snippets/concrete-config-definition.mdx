A **Concrete Config** is a specific set of parameter values that define the code's behavior for a given instance.
```yaml
speed: 10
features:
	spin: true
	jump: true
	backflip: false
accelerometer:
	id: acc_id_1
	offsets:
		x: 0.1
		y: 0.13
		z: -0.04
	scaling_factor:
		x: 1.0
		y: 1.02
		z: 0.99
```
