Navigate to the [service accounts](https://console.cloud.google.com/projectselector2/iam-admin/serviceaccounts) page in your Google Cloud Console. Select the Google Cloud project that you want to connect `miru` to. We're using **container-integration**. 

<img src="/images/integrations/gar/gcp-sa-proj-select.png" style={{borderRadius: '7px'}} />

<Note> To connect multiple Google Cloud projects, you will need to create a separate service account for each project you want to connect to `miru`. </Note>

Click the **Create Service Account** button to create a new service account for this Google Cloud Project.

<img src="/images/integrations/gar/gcp-create-sa.png" style={{borderRadius: '7px'}} />

Fill in the service account name and description before clicking **Create and Continue**. You can follow the naming convention we've used or choose your own.

<img src="/images/integrations/gar/gcp-fill-sa-details-form.png" style={{borderRadius: '7px'}} />

Finally, click **Done** to create the service account.
