From the command-line, use the install script or Homebrew to install the CLI.

<Tabs>
    <Tab title="Linux">
        ```bash
        curl -fsSL https://raw.githubusercontent.com/miruml/cli/main/install.sh | sh
        ```

        You can review the contents of the installation script [here](https://github.com/miruml/cli/blob/main/install.sh).
    </Tab>

    <Tab title="MacOS">
        ```bash
        brew install miruml/cli/miru
        ```
    </Tab>
</Tabs>

Verify the CLI is installed by invoking the help menu
```bash
miru --help
```
