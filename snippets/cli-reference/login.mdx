To begin using the CLI, you'll need to login.

```bash
miru login
```

Please retrieve your authentication token by following the
[link](https://configs.miruml.com/configs?cli_auth=true) below or by clicking on 
the key icon at the bottom left of the sidebar.

```
https://configs.miruml.com/configs?cli_auth=true
```

Copy your authentication token from the popup window.

<img src="/images/getting-started/cli-token.png" style={{borderRadius: '15px'}}/>

Once you've copied your authentication token, paste it into the CLI.
