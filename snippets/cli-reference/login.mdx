To begin using the CLI, you'll need to login.

```bash
miru login
```

You'll see a message in the terminal prompting you to get your authentication token.
Get your token by following the [link](https://configs.miruml.com/configs?cli_auth=true) 
or by clicking the key icon at the bottom of the sidebar.
