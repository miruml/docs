Navigate to the [API Keys page](https://configs.miruml.com/secrets/api-keys) in the dashboard and click **New API Key** in the top right corner.

Give the key a descriptive name and select the `devices:provision` scope. 

<Frame>
    ![](/images/secrets/api-keys/provision-scopes.png)
</Frame>

Finally, click **Create** to generate the key.

<Warning>
Copy your API key immediately after creation. Once you navigate away from this page, the key cannot be retrieved again.
</Warning>
