Navigate to the [API Keys page](https://configs.miruml.com/secrets/api-keys) in the dashboard. Click **New API Key** in the top right.

Provide a name for the API Key, select the `devices:provision` scope, and click **Create**.

<Frame>
    ![](/images/api-keys/provisioner.png)
</Frame>

<Warning>
Once an API key has been created, it can never be retrieved again. If you lose your API key, you must create a new one.
</Warning>
