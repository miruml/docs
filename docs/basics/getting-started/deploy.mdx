## SDK in Development
To begin using the Miru C++ SDK, navigate to the `/cpp` directory of the [getting-started](https://github.com/miruml/getting-started) repository cloned at the beginning of the guide.

Build the CMake project (you can find the build requirements [here](https://github.com/miruml/cpp-sdk?tab=readme-ov-file#build-requirements)).

```bash
cmake -B build -S .
```
```bash
cmake --build build
```

The [from_file](https://github.com/miruml/getting-started/tree/main/cpp/examples/from_file/main.cpp) example demonstrates how to use the SDK in development by loading a config instance from a file on the local file system.

```bash
./build/examples/from_file/from-file
```

The example simply reads in a config instance from the file system and lists the parameters inside. You should see the following output:

```bash [expandable]
$ ./build/examples/from_file/from-file
```

## SDK in Production
Navigate back to the `/cpp` directory of the [getting-started](https://github.com/miruml/getting-started) repository.

The [from_agent](https://github.com/miruml/getting-started/tree/main/cpp/examples/from_agent/main.cpp) example demonstrates how to use the SDK in production by loading a config instance from the Miru agent.

```bash
./build/examples/from_agent/from-agent
```

As before, the example simply reads in a config instance from the agent and lists the parameters inside. You should see the following output:

```bash [expandable]
$ ./build/examples/from_agent/from-agent
```
