## View a schema

To view a schema in Miru, navigate to the [Config Types page](https://app.mirurobotics.com/configs) and click into the config type that contains the schema you want to view.

<Frame>
  ![](https://assets.mirurobotics.com/docs/v02/images/config-schemas/list.png)
</Frame>

Clicking a schema opens the schema details panel, which displays information about the schema.

The **Content** tab contains the schema's content, while the **Metadata** tab contains the schema's metadata and its Git history.

## Delete a schema

Deleting a schema is irreversible. Once a schema is deleted, it cannot be recovered.

Deleting a config schema is only possible if there are no config instances for that schema. As such, most schemas cannot be deleted.

To delete a config schema, click the ellipses of the config schema you want to delete and select **Delete**.

<Frame>
  ![](https://assets.mirurobotics.com/docs/v02/images/config-schemas/ellipses-dropdown.png)
</Frame>

A confirmation dialog will appear. Hit **Delete** to confirm the deletion.

<Frame>
  ![](https://assets.mirurobotics.com/docs/v02/images/config-schemas/delete-dialog.png)
</Frame>
