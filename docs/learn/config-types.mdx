## Properties

<ParamField path="name" type="string">

<EditableBadge />

A human-readable display name that can contain most characters such as spaces, special characters, and punctuation. Must contain between 1 and 48 characters.

Examples: `Mobility`, `Safety Features`, `Perception`

</ParamField>

<ParamField path="slug" type="string">

<ImmutableBadge />

A unique, code-friendly identifier used to associate schemas with their config type. 

Slugs are immutable and must satisfy the following constraints:

- Contain only lowercase letters, digits, and hyphens (`-`)
- Start and end with a lowercase letter or digit
- Contain between 1 and 48 characters

Slugs are primarily used in code to reference config types in a human-readable format. For instance, config schemas are annotated with a config type slug to associate it with its proper config type.


Examples: `mobility`, `safety-features`, `perception`
</ParamField>

## Create a config type

Navigate to the [Config Types page](https://app.miruml.com/configs) and click **New Config Type** in the top right corner.

<Frame>
  ![](https://assets.miruml.com/docs/images/config-types/create-page.png)
</Frame>

Fill in the fields and hit **Create**.

<Frame width="100px">
  ![](https://assets.miruml.com/docs/images/config-types/create-dialog.png)
</Frame>

<Warning>
  Slugs cannot be modified after creation.
</Warning>

## Edit a config type

Click the ellipses (...) on the config type you want to edit and hit **Edit**.

<Frame >
  ![](https://assets.miruml.com/docs/images/config-types/ellipses-dropdown.png)
</Frame>

Update the desired fields and hit **Save**.

<Frame>
  ![](https://assets.miruml.com/docs/images/config-types/edit-dialog.png)
</Frame>

## Delete a config type

Deleting a config type is only possible if there exist no config instances for that config type. As such, most config types cannot be deleted.

If no config instances exist for the config type, deleting a config type removes all associated schemas.

<Warning>
  Deleting a config type cannot be undone.
</Warning>

To delete a config type, click the ellipses of the config type you want to delete and select **Delete**.

<Frame>
  ![](https://assets.miruml.com/docs/images/config-types/ellipses-dropdown.png)
</Frame>

A confirmation dialog will appear. Hit **Delete** to confirm the deletion.

<Frame>
  ![](https://assets.miruml.com/docs/images/config-types/delete-dialog.png)
</Frame>
