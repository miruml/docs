This quick-start guide walks through deploying your first configuration with Miru. Here's what we'll cover.

<Steps>
  <Step title="Create a config schema">
    We'll create the `Mobility` config type, install the [Miru CLI](/docs/references/cli/install), and push a schema to Miru.
  </Step>
  <Step title="Install the agent">
    Next, we'll install and activate the [Miru Agent](/docs/agent/intro) on your target device.
  </Step>
  <Step title="Deploy config instances">
    Finally, we'll deploy config instances and make some edits.
  </Step>
</Steps>
