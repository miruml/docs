To install the Miru Agent, please visit the [device provisioning](/docs/learn/devices/provision) section, which provides detailed instructions for installing and activating the agent via the following methods:

- [Dashboard](/docs/learn/devices/provision#dashboard) - a simple web interface for installing the agent on individual devices
- [API Keys](/docs/learn/devices/provision#api-keys) - a programmatic way to install the agent on many devices
- [Poor Connectivity](/docs/learn/devices/provision#poor-connectivity) - a method for installing the agent on devices with poor connectivity
