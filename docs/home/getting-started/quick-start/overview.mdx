This quick-start guide walks through deploying your first configuration with Miru. Here's what we'll cover.

<Steps>
  <Step title="Defining a config schema">
    First, we'll create the `Mobility` config type and push a schema from the command-line.
  </Step>
  <Step title="Installing the agent">
    Next, we'll install our target machine with the [Miru Agent](/docs/home/agent/intro), which is used to deploy config instances.
  </Step>
  <Step title="Deploying config instances">
    Finally, we'll deploy some config instances to our target machine and make some edits.
  </Step>
</Steps>
