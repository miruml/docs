## Overview
Some robots may have particularly poor connectivity. Whether located in a remote area, using a cellular connection, or otherwise, downloading the Miru Agent may be difficult if not untenable.

In these scenarios, we recommend downloading the Miru Agent on a computer with a stable connection, copying the Miru Agent to the robot, installing the agent manually, and then activating the agent (via the [Dashboard](/docs/home/agent/install/dashboard) or [API Keys](/docs/home/agent/install/api-keys)).
